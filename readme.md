# Lost In Forrest, Perdu en Forêt

Simple Rogue light, made with every simplest choice i can
Un jeux du type rogue, fait avec tous les choix les plus simple possible
https://raoul632.itch.io/lost-in-forest

### Installing

Cope Paste the File into Visual studio dont forget image directory 
Copiez les fichiers dans un projets visual studio ne pas oubliez le repertoire Image

##Description 
To Create Random Map, i just use a random walker 
Pour crée la carte aléatoire j'ai utilisé un marcheur aléatoire 

Pour L'entité composant systeme c'est juste de la composition un tableau qui peux contenir des objets differens 
l'entité n'étant plus qu'un numéros ;-), le systeme étant la parti du programme qui va lire les composants et les faire interargie entre eux 

For the entity component system it was just composition, an array who can contain different object. Entity was just a number and system 
read parse and interract with each component.
